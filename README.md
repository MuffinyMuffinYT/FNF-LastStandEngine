# Friday Night Funkin' - Last Stand Engine
Modified [Psych Engine](https://github.com/ShadowMario/FNF-PsychEngine) used for the Last Stand mod.

## Installation:
You must have [the most up-to-date version of Haxe](https://haxe.org/download/), seriously, stop using 4.1.5, it misses some stuff.
